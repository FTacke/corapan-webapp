### Technische Details

- **Alter Ansatz:** Docker Container mit blacklab-indexer (nicht verfügbar)
- **Neuer Ansatz:** Java -cp blacklab-server.jar nl.inl.blacklab.tools.IndexTool
- **Kanonischer Pfad:** data/blacklab_index.backup/tsv/*.tsv
- **Ziel-Index:** data/blacklab_index/ (Lucene 9.11.1)
- **Konfiguration:** config/blacklab/corapan-tsv.blf.yaml (17 Annotationen)

### Voraussetzungen (NEU)

- Java 11+ (für IndexTool)
- BlackLab JAR (User muss manuell herunterladen)
- TSV-Quelldaten in data/blacklab_index.backup/

### Breaking Changes

Index-Build benötigt jetzt:
1. Java 11+ im PATH
2. BlackLab JAR (Download: https://github.com/INL/BlackLab/releases)
3. JAR-Location: $env:BLACKLAB_JAR oder tools/blacklab/blacklab-server.jar

Docker-Container enthält NUR noch Server, KEINE Indexierungs-Tools.

### Nächste Schritte für Entwickler

1. BlackLab JAR herunterladen (siehe tools/blacklab/README.md)
2. Index neu aufbauen: .\scripts\build_blacklab_index.ps1
3. Tests durchführen (siehe Migration Guide, Test-Checkliste)
